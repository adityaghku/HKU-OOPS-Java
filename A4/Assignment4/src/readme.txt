1. Ensure that the relevant Java environment is installed and acccessible through the system terminal

2. Navigate to the directory containing Server.java Client.java files

3. Compile Server and Client using the commands

javac Server.java Client.java

4. Run the server using the command:

java Server

5. Run the game with two different terminals for player 1 and player 2

java Client
java Client

Whoever submits their name first is player 1 irrespective of which player opened the game first
