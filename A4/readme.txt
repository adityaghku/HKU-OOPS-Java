1. compile TicTacToeServer and TicTacToeClient

javac TicTacToeServer.java
javac TicTacToeClient.java


2. Run TicTacToeServer

java TicTacToeServer


3. Run TicTacToeClient with 2 instances of command prompt.

java TicTacToeClient
java TicTacToeClient

# Note that the player that submits his/her name first will be player 1. Regardless of the client opening order.
